## CONVERSOR KM PARA MILHAS / CELSIUS PARA FAHRENHEIT

Esta é uma aplicação que faz a conversão de um valor de celsius para fahrenheit e quilometros para milhas.
A aplicação exibe a mensagem com tres opçoes para o usuário. A primeira opção é a conversão do valor de celsius para fahrenheit, a segunda converte quilometros para milhas e a ultima encerra a aplicação. É usada uma estrutura condicional para executar ações de acordo com a opção escolhida, no caso são feitos cálculos nas opções 1 e 2.
Também é usada uma estrutura de repetição, usei a estrutura ENQUANTO, que roda a aplicação até o usuário escolher a opção 3 que encerra o programa.
