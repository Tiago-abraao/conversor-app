## CONVERSOR KM PARA MILHAS / CELSIUS PARA FAHRENHEIT

Esta é uma aplicação que faz a conversão de um valor de celsius para fahrenheit e quilometros para milhas.
A aplicação exibe a mensagem com tres opçoes para o usuário. A primeira opção é a conversão do valor de celsius para fahrenheit, a segunda converte quilometros para milhas e a ultima encerra a aplicação. Se o usuário escolher uma opção fora das 3 é exibida uma mensagem: OPÇÃO INVÁLIDA. É usada uma estrutura condicional para executar ações de acordo com a opção escolhida, no caso são feitos cálculos nas opções 1 e 2.
Também é usada uma estrutura de repetição, usei a estrutura ENQUANTO, que roda a aplicação até o usuário escolher a opção 3 que encerra o programa.

This is an application that converts values from Celsius to Fahrenheit and from kilometers to miles. The application displays a message with three options for the user. The first option converts a value from Celsius to Fahrenheit, the second converts kilometers to miles, and the last one exits the application. If the user chooses an option outside these three, a message is displayed: **INVALID OPTION**. A conditional structure is used to execute actions according to the selected option; in this case, calculations are performed in options 1 and 2. A repetition structure is also used — specifically a **WHILE loop**, which keeps the application running until the user selects option 3 to terminate the program.
